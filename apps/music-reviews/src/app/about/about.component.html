Over deze applicatie

Deze applicatie is gemaakt in het kader van het vak 'Clientside web frameworks' van Avans Hogeschool Breda. Github: https://github.com/TimdeLaater/MusicReviews/tree/main/apps/music-reviews

Naam van jouw project

Mijn applicatie gaat als volgt werken: Een user kan een account aanmaken en hier zijn/haar favorite albums opzoeken. Ook kunnen ze hun favorite artiesten opzoeken en de albums reviewen.

Entiteiten

De 3 entiteiten van mijn project zijn: -Artist, -Album, -Review.

User

  • naam: string
  • id: Number
  • email: string
  • birthdate: date
  • password: string

TrainingSchedule

  • naam: string
  • id: number
  • start date: date
  • amountOfDays: Number
  • active: boolean
  • training sesions: Training[]
  • total calories: Number
  • userId: number

Training

  • naam: string
  • id: number
  • length: number
  • total calories: Number
  • exercises: Exercises[]
  • finished: boolean
  • userId: number

Exercises

  • naam: string
  • id: Number
  • reps: Number
  • sets: Number
  • calories: Number
  • targeted muscles: string
  • equipmentNeeded: boolean
  • userId: number
